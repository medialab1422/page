Web Programming

This course introduces HyperText Markup Language (HTML) and Cascading Style Sheets (CSS).

Course Description

The course topics include:

  • 1. HTML syntax
  • 2. CSS syntax
  • 3. JavaScript
  • 4. HTML5 APIs syntactic features
  • 5. CSS3 features
  • 6. Document Object Model (DOM)
  • 7. jQuery library
  • 8. JavaScript Object Notation (JSON)
  • 9. Asynchronous JavaScript and XML (AJAX)
  • 10. Scalable Vector Graphics (SVG)
  • 11. HTML5 Canvas API
  • 12. WebGL
  • 13. Responsive Web Design (RWD)
  • 14. Bootstrap
  • 15. Web Typography
  • 16. Web Fonts
