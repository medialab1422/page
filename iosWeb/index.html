IOS 應用程式開發

This course introduces the application of Hypertext Markup Language (HTML) syntax and Cascading Style Sheets (CSS). The course content includes: Hypertext Markup Language Cascading Style Sheets JavaScript, an interpreted scripting language HTML5 APIs syntax CSS3 features Scalable Vector Graphics (SVG) HTML5 Canvas Application Programming Interface (API) WebGL for three-dimensional rendering。

iOS App development


這門課是關於如何使用多種標準技術來建構和設計網站。 1. 結構與內容（HTML） Hypertext Markup Language (HTML)：這是網頁的骨架。它教會你如何使用標籤（tags）來定義網頁的結構和內容，例如標題、段落、圖片、連結等。 HTML5 APIs 語法：介紹 HTML 的最新標準（HTML5）中新增的功能和語法，例如新的語義標籤（

等）以及各種應用程式介面。 HTML5 Canvas 應用程式介面 (API)：學習如何使用 元素來進行繪圖、製作圖表或基礎的動畫，這是一個強大的二維繪圖工具。 2. 樣式與視覺（CSS） Cascading Style Sheets (CSS)：這是網頁的外觀和設計。你將學習如何使用 CSS 來控制網頁元素的呈現方式，例如顏色、字體、佈局（排版）和響應式設計（讓網頁適應不同裝置）。 CSS3 特性：涵蓋 CSS 的最新版本（CSS3）所引入的強大功能，例如圓角、陰影、漸變、過渡（Transitions）、動畫（Animations）和彈性盒子（Flexbox）或格線（Grid）等先進佈局技術。 Scalable Vector Graphics (SVG)：學習一種基於 XML 的向量圖形格式。SVG 圖像在不失真的情況下可以任意縮放，非常適合用來顯示圖標、圖表和複雜的圖形。 3. 行為與互動（JavaScript） JavaScript (JS)：這是一種解釋型腳本語言，它是網頁的行為和互動性來源。學習如何使用 JavaScript 來實現動態效果、處理使用者輸入、與後端伺服器進行通訊等，是讓網頁「動起來」的關鍵。 4. 進階渲染與圖形 WebGL for 三維渲染：這是較為進階的主題，它允許網頁在不依賴任何外掛程式的情況下，透過瀏覽器的 Canvas 元素和 JavaScript 來渲染高性能的 3D 圖形。這在開發網頁遊戲或複雜的 3D 數據視覺化時非常重要。
